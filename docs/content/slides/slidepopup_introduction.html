Introduction to module: SlidePopup

The SlidePopup module is a simple but powerful module that allow one to fetch a slide at any time and display it on top of the active slideshow. The underlying slideshow is not affected and when the slidepopup is closed you return to the same state as before. The slide methods onEnter() and onExit() will be called when showing and hiding the popup.

It is possible to create as many slidepopups as needed.

Custom events

The SlidePopup module will dispatch the following events:

  • slidePopupLoad
  • slidePopupUnload

Popup content

The SlidePopup module will fetch its content in content/slides/ when calling the show instance method.

Limitations

It is not possible to open a slide that is already part of the active slideshow (the slideshow being viewed).

Constructor

A slide popup is create by calling new SlidePopup with an id paramater.

Examples

Example: creating a new slide popup by calling new Slidepopup.

The following example will create a new slide popup using the DOM element with id 'slidePopup' as container (the DOM element containing the loaded slide).

app.slidePopup = new SlidePopup('slidePopup');
