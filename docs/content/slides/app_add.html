app.add

Summary

Dynamically generate new slideshow or collection in presentation.

Core Presentation Method
Implemented in v1.0

Syntax

app.add(name, content, type)

Parameters

name string
Name of slideshow or collection.
content array
Array of slides or slideshows (slideshow, collection respectively)
type string
Type of content, possible values 'slideshow' [default], 'collection'

Description

app.add will create a new slideshow or collection based on existing slides or slideshows.

Calling app.add will not load any content, for that app.load should be used.

app.add is often used to create dynamic agendas, or to create dynamic slideshows based on user interactions.

Examples

Example: app.add to create a slideshow with name 'demo'

The following example will create the slideshow 'demo', making it available for loading

app.add('demo', ['first_slide', 'second_slide']);

Example: app.add to create a collection with name 'demo'

The following example will create the collection 'demo', making it available for loading

app.add('demo', ['section_one', 'section_two'], 'collection');
