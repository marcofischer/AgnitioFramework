Slidescroller.enable

Summary

Enable a direction of a Slidescroller.

Core Slidescroller Method
Implemented in v1.0

Syntax

app.scroller.enable(direction)

Parameters

direction string
The swipe direction to enable. Possible values: 'left', 'right', 'up', 'down'.

Description

Slidescroller.enable will enable a single swipe direction.

Examples

Example: app.scroller.enable to enable swiping to next slide.

The following example will enable swipe to next slide or section for a Slidescroller referenced through app.scroller.

app.scroller.enable('left');
