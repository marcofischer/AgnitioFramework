app.slideshow.removeEmbedded

Summary

Remove any embedded slideshows.

Core Slideshow Method
Implemented in v2.2

Syntax

app.slideshow.removeEmbedded()

Description

app.slideshow.removeEmbedded will remove any embedded slideshows from a slide.

app.slideshow.removeEmbedded is normally called from the slide's onExit method.

Examples

Example: app.slideshow.removeEmbedded to remove an embedded slideshow.

The following example will remove the embedded a slidehow with id "MoA" from the slide with id "efficacy".

	(function() {
	  var slide = app.slide.efficacy = {
	    elements: {
	      moa: "#moaContainer"
	    },
	    onEnter: function(ele) {
	      // Embed MoA slideshow with scroller
	      app.slideshow.embed('MoA', slide.element.moa, true);
	    },
	    onExit: function(ele) {
	    	// Remove embedded slideshows
	    	app.slideshow.removeEmbedded();
	    }
	  }
	})();
