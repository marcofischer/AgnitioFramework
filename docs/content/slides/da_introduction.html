Introduction to module: Dynamic Agenda

The dynamic agenda module provides the developer with an API that makes it possible for the user of the content to dynamically create new slideshows within the presentation. Those slideshows will be saved to the localStorage of the device for repeated use.

The dynamic agenda module comes bundled with a simple interface. However, it is possible to create any interface you wish using the API provided by the module.

Limitations

Currently the module will by default only create slideshows, not collections.

Initialize module

Add the following script tag to the index.html file:

<script src="modules/dynamic-agenda/dynamic-agenda.js"></script>

In setup.js:

app.agenda = new DynamicAgenda(name);

Parameters

name string
The name to be used in localStorage to store the agendas.

Examples

Example: Initializing the module

The following example would create a dynamic agenda that stores created slideshow in localStorage.DA_Placebo. This call would normally be placed in setup.js but can also be placed in the code setting up the interface for the dynamic agenda.

app.agenda = new DynamicAgenda('DA_Placebo');
