app.slideshow.get

Summary

Get the HTML element of a slide.

Core Slideshow Method
Implemented in v1.0

Syntax

app.slideshow.get(slideId)

Parameters

slideId string
The id of the slide. OPTIONAL

Description

app.slideshow.get will return the HTML element of a slide. If a parameter is given it will try to get the element corresponding to the id, and if no parameter is given then it will get the element of the current slide.

app.slideshow.get is a simple wrapper for document.getElementById(slideId).
