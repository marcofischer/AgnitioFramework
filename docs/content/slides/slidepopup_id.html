SlidePopup.id

Summary

Reference to the id of the SlidePopup container.

Module instance property
Implemented in v1.0

Syntax

app.slidePopup.id

Returns

Container id string
Id of the SlidePopup container.

Description

app.slidePopup.id will return the id for the container of a SlidePopup instance referenced through app.slidePopup. This id is supplied as a parameter to the constructor when creating the SlidePopup instance.
