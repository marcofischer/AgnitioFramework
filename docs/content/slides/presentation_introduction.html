Introduction to core module: Presentation

This JavaScript constructor initializes the presentation using the supplied configuration to create slideshows and collections. In the process it will ensure access to the global namespace app, and end by dispatching the presentationInit event.

Custom events

The Presentation module dispatches the following events

  • presentationInit

Presentation setup

The framework provides a setup.js file in the root of the app (since version 2.0). This file is used to configure and setup the presentation and modules.

Initializing the presentation

In order to run the presentation app.init() need to be run. This is normally done by including it last in the setup.js file.
