Util

Util is a JavaScript micro-library that provides some useful JavaScript utility functions.

Namespace

The library provides util as a global accessor.

Methods

Descriptions of methods are coming but the following are currently available:

  • bind(fn, me)
  • addClass(ele, classname)
  • hasClass(ele, classname)
  • removeClass(ele, classname)
  • swapClass(ele, classname1, classname2)
  • toggleClass(ele, classname)
  • openPDF(doc)
  • openPDF(doc)
  • getPosition(ele)
