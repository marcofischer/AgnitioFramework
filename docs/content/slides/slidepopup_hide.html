SlidePopup.hide

Summary

Hide and remove content of a slide popup.

Module instance method
Implemented in v1.0

Syntax

app.slidePopup.hide()

Description

SlidePopup.hide will hide the popup container and remove its content.

The slide's onExit method will be called when calling the hide method.

Examples

Example: app.slidePopup to hide a SlidePopup instance.

The following example will hide a SlidePopup instance referenced through app.slidePopup.

app.slidePopup.hide();
