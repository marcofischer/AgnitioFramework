Introduction to core module: Collection

The Collection module directly inherits from the Slideshow module but instead of keeping track of slides it keeps track of slideshows that are loaded together. The API methods are the same as for slideshows but they are accessed from the app.collection namespace.

The Collection constructor is used internally by the presentation and not called directly. To add a collection not defined in the configuration of the Presentation constructor, the method app.add should be used.

Custom events

The Collection module dispatches the following events:

  • collectionLoad
  • collectionUnload
  • sectionEnter
  • sectionExit
