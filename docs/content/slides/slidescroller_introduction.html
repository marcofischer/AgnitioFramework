Introduction to module: Slidescroller

The Slidescroller module listens for swipe events and will scroll to the appropriate slide or section. The constructor can be used to create as many scrollers as needed but in most presentations one should be enough.

This module can easily be replaced or removed if other behavior is desired, no core component of the framework requires it (although other custom modules may).

Convention

In Agnitio we use the following convention for creating our main scroller (for slideshows and sections): app.scroller = new Slidescroller();. In some of our modules, i.e. SlidePopup, we use the app.scroller object to disable and enable scrolling. When using Agnitio modules it is encouraged to follow the above convention.
