Presentation configuration

When initializing presentation in setup.js, it is possible to add some configuration options. You'll see a list below with possible values and an explanation for each.

Configuration options

type
Either dynamic [default] or json
orientation
Either landscape [default] or portrait
pathToSlides
Used to set custom path to slides, i.e. "content/slides/"
globalElements
Ids of DOM elements that should be accessible throughout presentation. Elements accessible through app.elements[id]
manageMemory
Either true or false [default]
slideshows
A JavaScript object mapping an array of slide ids to a slideshow name
collections
A JavaScript object mapping an array of slideshow ids to a collection name

Changelog
