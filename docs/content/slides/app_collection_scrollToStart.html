app.collection.scrollToStart

Summary

Navigate to the first section in the active collection.

Core collection Method
Implemented in v1.0

Syntax

app.collection.scrollToStart()

Parameters

Description

app.collection.scrollToStart will move to the first section in the active collection.

app.collection.scrollToStart is equivalent to app.collection.scrollTo(0).
