Introduction

This is the documentation to Agnitio's framework for creating interactive presenations on the iPad. It is built purely in HTML5, CSS and JavaScript so that it can easily be used by front-end developers using their favorite editors and tools.

New users are recommended to read through each link in the General Topics section.
