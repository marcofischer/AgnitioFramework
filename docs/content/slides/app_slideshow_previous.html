app.slideshow.previous

Summary

Navigate to the previous slide in the active slideshow.

Core Slideshow Method
Implemented in v1.0

Syntax

app.slideshow.previous()

Parameters

Description

app.slideshow.previous will move to the previous slide if one is available.

app.slideshow.previous use CSS3 transform (translate3d) to move the slideshow container. To update the speed with which slides shift, declare a CSS3 transition-duration on the slideshow class.
