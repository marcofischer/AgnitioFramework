app.elements

Summary

Object containing presentation DOM elements.

Core Presentation Property
Implemented in v1.0

Syntax

app.elements

Description

app.elements is a container object for DOM elements that should be accessible to the presentation.

By default the following elements (by id) are available:

  • presentation
  • slideshows

The Presentation configuration option globalElements will add elements to this object.
