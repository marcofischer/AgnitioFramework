app.slideshow.embed

Summary

Embed another slideshow within a slide.

Core Slideshow Method
Implemented in v2.2

Syntax

app.slideshow.embed(slideshowId, container, scroll)

Parameters

slideshowId string
Id of the slideshow to embed in the slide.
container DOM element
Element of the container where the slideshow should be inserted.
scroll boolean
If the a scroller (swipes) should be created for the embedded slideshow. OPTIONAL

Description

app.slideshow.embed allows one to embed another slideshow within the slide. Several slideshows can be embedded in a slide.

app.slideshow.embed is normally called from the slide's onEnter method.

NOTE: app.slideshow.removeEmbedded() need to be called to remove embedded slideshows in a slide. This is normally done from the slide's onExit method.

Examples

Example: app.slideshow.embed to embed a slideshow in a slide.

The following example will embed a slidehow with id "MoA" in the slide with id "efficacy".

	(function() {
	  var slide = app.slide.efficacy = {
	    elements: {
	      moa: "#moaContainer"
	    },
	    onEnter: function(ele) {
	      // Embedd MoA slideshow with scroller
	      app.slideshow.embed('MoA', slide.element.moa, true);
	    },
	    onExit: function(ele) {
	    	// Remove embedded slideshows
	    	app.slideshow.removeEmbedded();
	    }
	  }
	})();
