app.collection.getIndex

Summary

Get the order number of a slide.

Core Collection Method
Implemented in v1.0

Syntax

app.collection.getIndex(sectionId)

Parameters

sectionId string
The id of the section. OPTIONAL

Description

app.collection.getIndex will return the order number of a section. If a parameter is given it will try to get the order number corresponding to the id, and if no parameter is given then it will get the order number of the current section.

The order number is 0-based.
