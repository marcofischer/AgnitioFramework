Introduction to core module: Slideshow

Apart from the Presentation module this is the only module required in any presentation. Each presentation will contain at minimum one slideshow, but can contain as many as necessary. Collections are built by combining slideshows, where each slideshow becomes a section.

The Slideshow constructor is used internally by the presentation and not called directly. To add a slideshow not defined in the configuration of the Presentation constructor, the method app.add should be used.

Custom events

The Slideshow module dispatches the following events:

  • slideshowLoad
  • slideshowUnload
  • slideEnter
  • slideExit
