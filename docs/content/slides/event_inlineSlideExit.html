Event: inlineSlideExit

Summary

Event called when an inline slide is exited

Custom Event
Implemented in v1.0

Description

inlineSlideExit is dispatched every time a slide is exited in an inline slideshow. When the event is dispatched the property app.inline.current will contain the name of the slide being exited.

event.target will be the slide element. The name of the slide will be available through app.inline.current.

Note: The normal slideshow events slideEnter and slideExit will not be called when shifting slides in an inline slideshow. The slide methods onEnter() and onExit() will be called however.
