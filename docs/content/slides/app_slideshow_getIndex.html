app.slideshow.getIndex

Summary

Get the order number of a slide.

Core Slideshow Method
Implemented in v1.0

Syntax

app.slideshow.getIndex(slideId)

Parameters

slideId string
The id of the slide. OPTIONAL

Description

app.slideshow.getIndex will return the order number of a slide. If a parameter is given it will try to get the order number corresponding to the id, and if no parameter is given then it will get the order number of the current slide.

The order number is 0-based.
