app.slideshow.next

Summary

Navigate to the next slide in the active slideshow.

Core Slideshow Method
Implemented in v1.0

Syntax

app.slideshow.next()

Parameters

Description

app.slideshow.next will move to the next slide if one is available.

app.slideshow.next use CSS3 transform (translate3d) to move the slideshow container. To update the speed with which slides shift, declare a CSS3 transition-duration on the slideshow class.
