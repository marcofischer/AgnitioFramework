Event: slideshowUnload

Summary

Event called before a slideshow is unloaded

Custom Event
Implemented in v1.0

Description

slideshowUnload is dispatched every time a slideshow is unloaded. When the event is dispatched the namespace app.slideshow is still available and refers to the slideshow that is unloading.

slideshowUnload is automatically called when loading another slideshow, or collection.

event.target will be the slideshow element. This means that you can use event.target.id to get the name of the unloading slideshow in the event handler.
