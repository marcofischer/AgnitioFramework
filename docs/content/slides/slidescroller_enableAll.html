Slidescroller.enableAll

Summary

Enable a Slidescroller.

Core Slidescroller Method
Implemented in v1.0

Syntax

app.scroller.enableAll()

Description

Slidescroller.enableAll will enable all swipe directions of a Slidescroller.

Examples

Example: app.scroller.enableAll to enable swiping to any neighboring slide.

The following example will enable all swipes for a Slidescroller referenced through app.scroller.

app.scroller.enableAll();
