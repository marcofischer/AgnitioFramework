SlidePopup.show

Summary

Add and show content of a slide popup.

Module instance method
Implemented in v1.0

Syntax

app.slidePopup.show(slide)

Parameters

slide string
Name of the slide to be displayed in popup.

Description

SlidePopup.show will fetch slide content and display the popup container.

The slide's onEnter method will be called when calling the show method.

Examples

Example: app.slidePopup to show a SlidePopup instance.

The following example will get the content of the slide 'efficacy' and display a SlidePopup instance referenced through app.slidePopup.

app.slidePopup.show('efficacy');
