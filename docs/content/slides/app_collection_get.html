app.collection.get

Summary

Get the HTML element of a slide.

Core Collection Method
Implemented in v1.0

Syntax

app.collection.get(sectionId)

Parameters

sectionId string
The id of the section. OPTIONAL

Description

app.collection.get will return the HTML element of a section. If a parameter is given it will try to get the element corresponding to the id, and if no parameter is given then it will get the element of the current section.

app.collection.get is a simple wrapper for document.getElementById(sectionId).
