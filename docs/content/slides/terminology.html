Presentation structures (terminology)

All pages in a presentation are referred to as slides, and those slides are always part of a slideshow, but slideshows can optionally be combined into collections where each slideshow then become a section (chapter) in that collection.

An inline slideshow is a slideshow displayed inside of a slide.

Sections of a collection are often referred to as chapters.

Core modules are the inner workings of the framework, and are only edited by framework developers. The Slideshow core module is an example.

Modules are packaged code with a specific functionality beyond the scope of a single slide. An example is a thumbnail navigation module.

Assets are reusable code affecting a single slide, i.e. an animated pie chart.
