Draggy

Draggy is a JavaScript microlibrary for moving elements in Webkit browsers, works both on desktop and iOS devices.

Constructor Syntax

app.dragObj = new Draggy(attachTo, config);

Parameters

attachTo DOM element or id
The element, or the id of the element, receiving the drag behavior.
config object (optional)
Configuration options for the draggy object.

Config

onChange function
A function called when object is dragged. Takes x and y coordinates as parameters.
limitsX array (defaults to [-9999, 9999])
Range of possible x-values. The draggy object cannot be moved outside of these.
limitsY array (defaults to [-9999, 9999])
Range of possible y-values. The draggy object cannot be moved outside of these.
restrictX boolean (defaults to false)
If set to true the draggy object cannot be moved in horizontal direction.
restrictY boolean (defaults to false)
If set to true the draggy object cannot be moved in vertical direction.

Custom events

The Draggy library will dispatch custom events when element is dragged or dropped.

onDrag
Dispatched on mousemove or touchmove. event.target will be the draggy element.
onDrop
Dispatched on mouseup or touchend. event.target will be the draggy element.
