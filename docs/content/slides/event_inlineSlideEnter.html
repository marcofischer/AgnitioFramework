Event: inlineSlideEnter

Summary

Event called when an inline slide is entered

Custom Event
Implemented in v1.0

Description

inlineSlideEnter is dispatched every time a slide is entered in an inline slideshow. When the event is dispatched the property app.inline.current will contain the name of the entered slide.

event.target will be the slide element. The name of the inline slide will be available through app.inline.current.

Note: The normal slideshow events slideEnter and slideExit will not be called when shifting slides in an inline slideshow. The slide methods onEnter() and onExit() will be called however.
