DynamicAgenda.contentMap

Summary

Object mapping names of agendas to automatically generated ids.

It is used internally to translate between the user defined name of an agenda to the generated id that is used to store it in localStorage.

Module Property
Implemented in v1.0

Syntax

app.agenda.contentMap

Description

app.agenda.contentMap can be used to find the id that is used to reference the agenda in localStorage.

The module will automatically create an id for each agenda in the format of DA_[timestamp in ms] (i.e. DA_1351241161034).
