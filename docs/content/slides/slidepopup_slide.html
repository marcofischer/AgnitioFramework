SlidePopup.slide

Summary

Id of slide loaded in popup.

Module instance property
Implemented in v1.0

Syntax

app.slidePopup.slide

Returns

Slide id string
Slide id of displayed slide. Defaults to null.

Description

app.slidePopup.slide will return the slide id of the slide currently loaded in the slide popup referenced through app.slidePopup. If no slide is loaded it will return null.
