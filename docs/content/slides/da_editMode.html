DynamicAgenda.editMode

Summary

Property returning true or false depending on if the module is currently in edit mode or not.

Module Property
Implemented in v1.0

Syntax

app.agenda.editMode

Description

app.agenda.editMode will return true if the module currently has an agenda open for editing (app.agenda.edit(name) has been called).

This is a property that is very useful when building an interface for the agenda. It can be used to check if we are in edit mode, and if so update interface with the name and selected slides. If we are not in edit mode you will probably want to call app.agenda.create(); instead.
