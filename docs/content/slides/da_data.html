DynamicAgenda.data

Summary

Object in memory holding all information about the stored agendas.

Module Property
Implemented in v1.0

Syntax

app.agenda.data

Description

app.agenda.data is a representation of the data in localStorage, and any agenda currently being worked on.

The following is an example of an object stored in app.agenda.data:

"DA_1351241161034" : {
	name: "My agenda",
	slides: ["intro", "efficacy", "studies", "survey"],
	type: "slideshow"
}

Examples

Example: Getting the data for a specific agenda

The following example will get the data for an agenda with the name "Demo agenda".

var id = app.agenda.contentMap['Demo agenda']; // returns the id of the agenda
var data = app.agenda.data[id];

Example: Getting the data for the current agenda

The following example will get the data for an agenda currently being worked on (edited). This will be used when building interfaces

var current = app.agenda.current; // returns the id of the current agenda
var data = app.agenda.data[current];
