app.slideshow.scrollToStart

Summary

Navigate to the first slide in the active slideshow.

Core Slideshow Method
Implemented in v1.0

Syntax

app.slideshow.scrollToStart()

Parameters

Description

app.slideshow.scrollToStart will move to the first slide in the active slideshow.

app.slideshow.scrollToStart is equivalent to app.slideshow.scrollTo(0).
