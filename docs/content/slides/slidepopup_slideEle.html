SlidePopup.slideEle

Summary

DOM element of slide loaded in popup.

Module instance property
Implemented in v1.0

Syntax

app.slidePopup.slideEle

Returns

Slide element DOM element
Slide element of displayed slide. Defaults to null.

Description

app.slidePopup.slideEle will return the DOM element of the slide currently loaded in the slide popup referenced through app.slidePopup. If no slide is loaded it will return null.
