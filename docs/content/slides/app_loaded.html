app.loaded

Summary

Reference to the currently loaded content object (slideshow or collection).

Core Presentation Property
Implemented in v1.0

Syntax

app.loaded

Description

app.loaded will give you access to the loaded top object (slideshow or collection).

app.loaded will be the same as either the currently loaded slideshow, app.slideshow, or collection, app.collection, and is there as a convenience so that it is not necessary to check the type of the loaded content.
