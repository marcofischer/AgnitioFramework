Touchy

Touchy is a JavaScript micro-library that will make it easy to have presentations work on both desktop and iOS devices.

Custom events

The Touchy library will dispatch custom events when user interactions take place. By listening to these events it is possible to run code when those interactions take place (on desktop and iPad).

tap
Dispatched on mouseup or touchend. Used instead of click.
doubleTap
Dispatched when two tap events take place within 500ms of each other. Used instead of doubleClick.
longTouch
Dispatched when holding the finger or mouse (left button) down, without moving, for more than 500ms.
swipeLeft
Dispatched when moving finger or mouse (left button down) left over the content for less than 500ms.
swipeRight
Dispatched when moving finger or mouse (left button down) right over the content for less than 500ms.
swipeUp
Dispatched when moving finger or mouse (left button down) up over the content for less than 500ms.
swipeDown
Dispatched when moving finger or mouse (left button down) down over the content for less than 500ms.
drag
Dispatched when moving finger or mouse (left button down) over the screen for more than 500ms.
twoFingerTap
Touch event only! Dispatched on touchend when two fingers are touching screen.
