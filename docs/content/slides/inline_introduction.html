Introduction to core module: InlineSlideshow

The InlineSlideshow module directly inherits from the Slideshow module but instead of adding its content to the main slideshow element, it will add the slideshow to a slide in the currently running slideshow. The API methods are the same as for slideshows but they are accessed from the app.inline namespace.

The InlineSlideshow constructor is used to manually create each inline slideshow.

Custom events

The InlineSlideshow module dispatches the following events:

  • inlineSlideshowLoad
  • inlineSlideshowUnload
  • inlineSlideEnter
  • inlineSlideExit
