app.collection.previous

Summary

Navigate to the previous section in the active collection.

Core Collection Method
Implemented in v1.0

Syntax

app.collection.previous()

Parameters

Description

app.collection.previous will move to the previous section if one is available.

app.collection.previous use CSS3 transform (translate3d) to move the collection container. To update the speed with which section shift, declare a CSS3 transition-duration on the collection class.
