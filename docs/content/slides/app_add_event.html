app.addEvent

Summary

Add slide-specific event listener, which will be automatically cleaned up.

Core Presentation Method
Implemented in v2.1

Syntax

app.addEvent(eventType, callback, attachTo)

Parameters

eventType string
Type of event to listen to.
callback function
Function to call when event is fired.
attachTo element
DOM element which the listener should be added to. Defaults to document

Description

app.addEvent should be used when adding event listeners to slides. The framework will add an event listener which will be automatically cleaned up when exiting the slide.

Examples

Example: app.addEvent to add an event listener

The following example will add an event listener for a click on a button.

app.addEvent('tap', this.handleBtnClick, this.demoBtn);
