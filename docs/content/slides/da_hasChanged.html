DynamicAgenda.hasChanged

Summary

Property returning true or false depending on if there has been any changes applied to the agenda currently being worked on.

Module Property
Implemented in v1.0

Syntax

app.agenda.hasChanged

Description

app.agenda.hasChanged will return true if there has been any unsaved changes to the agenda currently being worked on.

This is a property that is very useful when building an interface for the agenda. It can for example be used to display a confirmation alert when exiting an agenda without saving.
