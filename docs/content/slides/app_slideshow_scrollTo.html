app.slideshow.scrollTo

Summary

Navigate to a slide in the active slideshow.

Core Slideshow Method
Implemented in v1.0

Syntax

app.slideshow.scrollTo(slide)

Parameters

slide string/number
Id or number (order) of the slide.

Description

app.slideshow.scrollTo will move to the slide supplied as parameter, if it is available. It will only navigate within a slideshow, for navigating to slides in other slideshows use app.goTo.

app.slideshow.scrollTo accepts either the slide id, or the slide order number (in the active slideshow) as parameter.

Examples

Example: app.slideshow.scrollTo to navigate to a slide using its id.

The following example will move to the slide with the id 'third_slide' if it is available in the active slideshow.

app.slideshow.scrollTo('third_slide');

Example: app.slideshow.scrollTo to navigate to a slide using its order number.

The following example will move to the fourth slide in the active slideshow (ordering is 0-based).

app.slideshow.scrollTo(5);
