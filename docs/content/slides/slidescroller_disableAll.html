Slidescroller.disableAll

Summary

Disable a Slidescroller.

Core Slidescroller Method
Implemented in v1.0

Syntax

app.scroller.disableAll()

Description

Slidescroller.disableAll will disable all swipe directions of a Slidescroller.

Examples

Example: app.scroller.disableAll to disable swiping to any neighboring slide.

The following example will disable all swipes for a Slidescroller referenced through app.scroller.

app.scroller.disableAll();
