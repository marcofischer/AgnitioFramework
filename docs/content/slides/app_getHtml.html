app.getHtml

Summary

Get HTML markup as a string.

Core Presentation Method
Implemented in v1.0

Syntax

app.getHtml(name, path, callback)

Parameters

name string
Name of file excluding file extension.
path string
Path to folder where file is located relative to index file. Defaults to 'content/slides/'.
callback function
Function to be called if retrieval of HTML is successful.

Description

app.getHtml will use a XMLHttpRequest to read the markup of a HTML file.

app.getHtml can be used to retrieve any HTML markup from anywhere within the presentation. This makes it easy to load content on-demand.

app.getHtml is used internally by the framework to get markup of the slides when a slideshow or collection is loaded.

Examples

Example: app.getHtml to fetch the markup of slide 'content/popups/extras.html'

The following example will retrieve the markup of extras.html and save it (to localStorage) as a string.

app.getHtml('extras',
            'content/popups/',
            function(data) { localStorage.extras = data; });
