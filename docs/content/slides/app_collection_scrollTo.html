app.collection.scrollTo

Summary

Navigate to a section in the active collection.

Core Collection Method
Implemented in v1.0

Syntax

app.collection.scrollTo(slide)

Parameters

slide string/number
Id or number (order) of the section.

Description

app.collection.scrollTo will move to the section supplied as parameter, if it is available. It will only navigate within a collection, for navigating to sections in other collections use app.goTo.

app.collection.scrollTo accepts either the section id, or the section order number (in the active collection) as parameter.

Examples

Example: app.collection.scrollTo to navigate to a section using its id.

The following example will move to the section with the id 'chapter_3' if it is available in the active collection.

app.collection.scrollTo('chapter_3');

Example: app.collection.scrollTo to navigate to a section using its order number.

The following example will move to the fourth section in the active collection (ordering is 0-based).

app.collection.scrollTo(5);
