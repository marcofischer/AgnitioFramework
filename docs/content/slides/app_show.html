app.show

Summary

Show an array of slides (temporary slideshow).

Core Presentation Method
Implemented in v2.0

Syntax

app.show(content)

Parameters

content array or string
Slide or slides to show.

Description

app.show will load the slides supplied as a parameter in a temporary slideshow. Good for development or showing dynamic previews.

Examples

Example: app.show of three slides

The following example will load a temporary slideshow with three slides.

app.show(['intro', 'efficacy', 'summary']);
