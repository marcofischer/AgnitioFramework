Slidescroller.disable

Summary

Disable a direction of a Slidescroller.

Core Slidescroller Method
Implemented in v1.0

Syntax

app.scroller.disable(direction)

Parameters

direction string
The swipe direction to disable. Possible values: 'left', 'right', 'up', 'down'.

Description

Slidescroller.disable will disable a single swipe direction.

Examples

Example: app.scroller.disable to disable swiping to next slide.

The following example will disable swipe to next slide or section for a Slidescroller referenced through app.scroller.

app.scroller.disable('left');
