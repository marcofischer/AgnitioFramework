app.load

Summary

Loads a slideshow or collection into current view.

Core Presentation Method
Implemented in v1.0

Syntax

app.load(name, skipScroll)

Parameters

name string
Name of slideshow or collection.
skipScroll boolean (defaults to false)
If to skip scrolling to the first slide of the slideshow or collection.

Description

app.load will replace the current content in view with the slideshow or collection given as parameter.

Calling app.load with an undefined slideshow- or collection name will unload current content and generate a JavaScript error.

app.load is used by app.init when loading the presentation.

Examples

Example: app.load of a slideshow with name 'demo'

The following example will open the slideshow 'demo' if it exist, or break the presentation if it doesn't

app.load('demo');

Changelog

Version number refers to framework version.

2.0
The second parameter type was removed and skipScroll was added. As type is now handled internally in the framework this change will not break code built on version 1.0 of the framework.
