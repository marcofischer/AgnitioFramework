app.slide

Summary

An object containing slide methods for each slide.

Core Presentation Property
Implemented in v1.0

Syntax

app.slide

Description

app.slide will give you access an object with the same name as a slide id, containing methods specific to that slide.

app.slide by default contain two mandatory methods for each slide: onEnter and onExit. By default these are empty but can be overwritten with code that should be run each time a slide is entered and exited respectively.
