Introduction to module: Thumbnails

The thumbnails module will allow you to easily create thumbnails navigation for both slideshows or collections.

The Thumbnails constructor will allow the creation of as many thumbnails navigation menus as the presentation requires.

Module
Implemented in v2.4.0

Requirements

  • draggy.js
  • touchy.js
  • util.js

Constructor Syntax

app.thumbs = new Thumbnails(config)

Parameters

config object
Configuration of the thumbnails.

Configuration options

attachTo string (optional)
Name of a slideshow or collection to activate the thumbnails menu when loaded.
container string (optional)
Id of the DOM element to which the menu markup should be added. Defaults to 'thumbs'.
pathToThumbs string (optional)
Path to the thumbnail images to be used. Defaults to 'content/img/thumbs/<id>.png'.
hideSingles boolean (optional)
Used if it shouldn't display if there is a single slide. Defaults to false.

Examples

Example: Calling new Thumbnails constructor

The following example will create a thumbnails menu (global).

app.thumbs = new Thumbnails();

Example: Calling new Thumbnails constructor for specific content

The following example will create a Thumbnail menu for the 'efficacy' slideshow.

app.efficacyThumbs = new Thumbnails({
  attachTo: "efficacy"
});

Example: Calling new Thumbnails constructor for 'placebo' content.

The following example will create a thumbnails menu for the 'placebo' collection, using a custom path to the thumbnail images. Also, thumbnail will not be displayed if where there is only a single slide available.

app.placeboThumbs = new Thumbnails({
  attachTo: 'placebo',
  pathToThumbs: 'content/thumbs/<id>.jpg',
  hideSingles: true
});
