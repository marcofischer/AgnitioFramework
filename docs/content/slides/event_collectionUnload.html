Event: collectionUnload

Summary

Event called before a collection is unloaded

Custom Event
Implemented in v1.0

Description

collectionUnload is dispatched every time a collection is unloaded. When the event is dispatched the namespace app.collection is still available and refers to the collection that is unloading.

collectionUnload is automatically called when loading another collection, or collection.

event.target will be the collection element. This means that you can use event.target.id to get the name of the unloading collection in the event handler.
