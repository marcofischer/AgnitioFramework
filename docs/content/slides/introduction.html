Framework introduction

The framework provides organization, libraries, and most importantly an API for building several types of presentations/slideshows. It is modular, meaning that it is easy to plug in and reuse modules/assets. The modular nature and folder/file organization also makes it easy for several developers working on the same presentation at the same time.

Examples of presentation types

  • Simple slideshow
  • Multiple slideshows
  • Collection with sections
  • Multiple slideshows with and without sections
  • Dynamic slideshows
