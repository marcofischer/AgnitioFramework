Introduction to module: AutoMenu

The AutoMenu module will allow you to easily create a presentation, slideshow, or collection menu that is automatically populated with links.

Core Module
Implemented in v2.1

Requirements

  • Presentation type "json"
  • draggy.js
  • touchy.js
  • util.js

Constructor Syntax

app.menu = new AutoMenu(config)

Parameters

config object
Configuration of the menu.

Configuration options

attachTo array/string
Names of slideshows or collections to activate the menu when loaded.
offset integer (optional)
Number of pixels to offset the menu (i.e. to hide links to the left).
offsetLinks integer (optional)
Number of links to offset the menu in default state. Used instead of offset.
prepend array (optional)
Objects of links to prepend to the menu.
append array (optional)
Objects of links to append to the menu.
links object (optional)
Object used to configure links, i.e. to change the link text or class.

Examples

Example: Calling new AutoMenu constructor for presentation menu

The following example will create a presentation menu with no offset. It will run if placebo or efficacy structures are loaded.

app.menu = new AutoMenu({
  attachTo: ['placebo', 'efficacy'],
  links: {
    intro: {title:" ", className:"home"},
    conclusion: {title:"Summary"}
  }
});
