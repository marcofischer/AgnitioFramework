Introduction to module: Menu

The menu module will allow you to easily create a presentation, slideshow, or collection menu.

The Menu constructor will allow the creation of as many menus as the presentation requires.

Requirements

  • draggy.js
  • touchy.js
  • util.js

Constructor Syntax

app.menu = new Menu(config)

Parameters

config object
Configuration of the menu.

Configuration options

attachTo string (optional)
Name of a slideshow or collection to activate the menu when loaded. Leave off if global menu.
container string (optional)
Id of the DOM element to which the menu markup should be added. Defaults to 'mainmenu'.
offset integer (optional)
Number of pixels to offset the menu in default state. Used, for example, to hide menu items left of the home link.
links array
Array with data about menu items. Each item has at least a title and a goTo property.

Examples

Example: Calling new Menu constructor for presentation menu

The following example will create a presentation menu (global) with no offset.

app.menu = new Menu({
  links: [
    { title:"Home", goTo:"intro", className:"home" },
    { title:"Efficacy", goTo:"efficacy" },
    { title:"Patient Profiling", goTo:"patient_profiling" },
    { title:"References", goTo:"refs" }
  ]});

Example: Calling new Menu constructor for slideshow menu

The following example will create a slideshow menu for the 'efficacy' slideshow.

NOTE: the goTo property takes dot-separated path to the content, i.e. efficacy.studies will navigate to the studies slide in the efficacy slideshow.

app.efficacyMenu = new Menu({
  attachTo: "efficacy",
  links: [
    { title:"Intro", goTo:"efficacy.intro" },
    { title:"First line", goTo:"efficacy.firstline" },
    { title:"Studies", goTo:"efficacy.studies" },
    { title:"Results", goTo:"efficacy.results" }
  ]
});

Example: Calling new Menu constructor for collection menu

The following example will create a collection menu for the 'placebo' collection, with an offset of -162 pixels (to hide the reference menu item by default).

NOTE: the goTo property takes dot-separated path to the content, i.e. placebo.efficacy.studies will navigate to the studies slide in the efficacy section of the placebo collection.

app.placeboMenu = new Menu({
  attachTo: 'placebo',
  offset: -162,
  links: [
    { title:"References", goTo:"placebo.references" },
    { title:"Intro", goTo:"placebo.intro", className:"home" },
    { title:"Efficacy", goTo:"placebo.efficacy" },
    { title:"MoA", goTo:"placebo.moa },
    { title:"References", goTo:"placebo.refs" }
  ]
});
