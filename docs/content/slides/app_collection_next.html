app.collection.next

Summary

Navigate to the next section in the active collection.

Core collection Method
Implemented in v1.0

Syntax

app.collection.next()

Parameters

Description

app.collection.next will move to the next section if one is available.

app.collection.next use CSS3 transform (translate3d) to move the collection container. To update the speed with which section shift, declare a CSS3 transition-duration on the collection class.
