Event: contentLoad

Summary

Event called whenever a slideshow or collection has loaded

Custom Event
Implemented in v1.0

Description

contentLoad is dispatched every time a slideshow or collection is loaded. This event should be used in modules that should work for both slideshows and collections. If a module is specific to one type the specific event should be used, i.e. slideshowLoad.

event.target will be the slideshow or collection element. This means that you can use event.target.id to get the name of the loaded slideshow or collection in the event handler.
