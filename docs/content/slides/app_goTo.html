app.goTo

Summary

Go to a location in the presentation.

Core Presentation Method
Implemented in v1.0

Syntax

app.goTo(name, content, subcontent)

Parameters

name string
Name of slideshow or collection.
content string
Name of slide or section (slideshow).
subcontent string
Name of slide (if going to a collection).

Description

app.goTo will allow you to navigate to anywhere within a presentation.

app.goTo will first load a slideshow (if not already loaded), and then scroll to correct slide. This means that onEnter() and onExit() will be called on the first slide of the slideshow.

Examples

Example: app.goTo to navigate to slide 'third_slide' in 'demo' slideshow.

The following example will load the slideshow 'demo', and then scroll to the slide with the name 'third_slide'.

app.goTo('demo', 'third_slide');

Example: app.goTo to navigate to slide 'third_slide' in 'chapter_4' section, of the 'demo' collection.

The following example will load the collection 'demo', then scroll to the section 'chapter_4', and then scroll to the slide with the name 'third_slide'.

app.goTo('demo', 'chapter_4', 'third_slide');
